# Masternode

| **Chủ đề**                                                                                                | **Nội dung**                                                                                                                                    |
| --------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| [Yêu cầu](https://docs.viction.xyz/masternode/requirements)                                               | Những khuyến nghị cơ bản để trở thành ứng cử viên Masternode (Masternode Candidate).                                                            |
| [Chạy full node](https://docs.viction.xyz/masternode/run-a-full-node)                                     | Hướng dẫn cách chạy Viction Masternode trên Testnet và Mainnet mà không cần sử dụng Docker và tmn.                                              |
| [Đăng ký Node của bạn](https://docs.viction.xyz/masternode/apply-your-node)                               | Sau khi full node được thiết lập và chạy, tài liệu này sẽ hướng dẫn bạn cách trở thành Masternode chính thức.                                   |
| [Cơ chế Slashing trên Viction](https://docs.viction.xyz/masternode/viction-slashing-mechanism)            | Nếu Masternode không ký xác thực bất kỳ block nào trong toàn bộ epoch, Masternode sẽ bị loại khỏi danh sách Masternode trong 5 epoch tiếp theo. |
| [Snapshot dữ liệu chuỗi (Chain Data Snapshots)](https://docs.viction.xyz/masternode/chain-data-snapshots) | Snapshot là một bản sao chính xác ghi lại trạng thái của blockchain tại một block height cụ thể.                                                |
